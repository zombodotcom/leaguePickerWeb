üöÄ Installation Instructions

1

Download the Extension

Click the "Download Extension" button above to get the extension files

2

Extract the Files

Extract the downloaded ZIP file to a folder on your computer

3

Open Browser Extensions

Chrome/Edge: Navigate to chrome://extensions/
Firefox: Navigate to about:debugging

4

Enable Developer Mode

Chrome/Edge: Toggle "Developer mode" in the top right corner
Firefox: Click "This Firefox" in the left sidebar

5

Load the Extension

Chrome/Edge: Click "Load unpacked" and select the extension folder
Firefox: Click "Load Temporary Add-on" and select the manifest.json file

6

Start Using!

Click the extension icon in your browser toolbar and upload your lockfile
